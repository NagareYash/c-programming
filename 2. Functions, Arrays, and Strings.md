## Preprocessor Directives
Preprocessor directives are commands that begin with `#` and are processed before the actual compilation begins. They are not C statements and do not end with a semicolon.
### Common Uses
**Including Header Files:**
```c
#include <stdio.h>      // System header file
#include "myfile.h"     // User-defined header file
```

**Difference between `< >` and `" "`:**
- `< >` - Looks in system directories (like `/usr/include`) but NOT the current directory​
- `" "` - Looks in the current directory first, then system directories​
## Functions
Functions are the essence of C programming. They provide a convenient way to encapsulate computation and can be reused throughout your program. Everything in C is essentially a function.
A C program consists of one or more functions, with at least one function called `main()`, which is the entry point of your code.​
### Why Use Functions?
- Break down complex problems into smaller, manageable pieces
- Reuse code without writing it multiple times
- Make ograms easier to read and maintain
- Test individual parts of code separately
### Syntax
```c
return-type function-name(parameters) {
    declarations;
    statements;
    return value;  // if any
}
```
### Example
1. Method 1 of defining a functions
```c
#include <stdio.h>

int square(int num) {
    int j;
    j = num * num;
    return j;
}

int main() {
    int a, b;
    printf("Enter an integer: ");
    scanf("%d", &a);
    b = square(a);     // Call the function
    printf("Square = %d\n", b);
    return 0;
}
```
2. Method 2 of defining a function
```c
#include <stdio.h>

int square(int);    // Function prototype

int main() {
    int a, b;
    printf("Enter an integer: ");
    scanf("%d", &a);
    b = square(a);     // Call the function
    printf("Square = %d\n", b);
    return 0;
}

int square(int num) {
    int j;
    j = num * num;
    return j;
}
```
## Call by Value
When you pass a variable to a function in C, the function receives a **copy** of that variable, not the original. Think of it like photocopying a document - you can write on the copy, but the original stays unchanged.​
### Example
```c
#include <stdio.h>

void changeNumber(int x) {
    printf("Inside function before change: %d\n", x);
    x = 100;
    printf("Inside function after change: %d\n", x);
}

int main() {
    int num = 5;
    
    printf("Before calling function: %d\n", num);
    changeNumber(num);
    printf("After calling function: %d\n", num);
    
    return 0;
}
```

## Arrays
Arrays are used when we need many variables of the same type.
### Declaration and Initialization
```c
int grades[10];                          // Declare array
int numbers[5] = {10, 20, 30, 40, 50};  // Declare with values
int scores[] = {85, 90, 95};             // Size auto-calculated
```
**Important:** Array indices start at 0.​
### Accessing Elements
```c
int grades[5] = {55, 63, 78, 89, 95};

printf("%d\n", grades[0]);    // Prints 55
grades[2] = 80;               // Changes third element
```
### Memory Representation
`int grades[5] = {55, 63, 78, 89, 95};`

| Position | 0   | 1   | 2   | 3   | 4   |
| -------- | --- | --- | --- | --- | --- |
| Content  | 55  | 63  | 78  | 89  | 95  |
## Strings
A string is an array of characters. In C, strings always end with a null terminator `\0`
### Declaration
```c
char name[20];                  // Declare with size
char message[] = "Hello";       // Size auto-calculated
char text[50] = "Welcome";      // Partially filled
```
### String stored in Memory
`char str[] = "fox";`

| Position |  0  |  1  |  2  |  3  |
| :------: | :-: | :-: | :-: | :-: |
| Content  |  f  |  o  |  x  | \0  |

| `strncat()` |        Appends n characters         | `strncat(str1, str2, n);` |
| `strcmp()`  |        Compares two strings         |    Returns 0 if equal     |
| `strchr()`  | Finds first occurrence of character |      Returns pointer      |
| `strstr()`  | Finds first occurrence of substring |      Returns pointer      |

## Character Input/Output
`getchar()` is a standard library function that takes a single input character from standard input whereas `putchar()` writes a single character to the standard output i.e. console/terminal.

```c
char c; 
c = getchar();      // Read one character 
putchar(c);         // Print one character
```
### Example
```c
#include <stdio.h>

int main() {
    int myChar;
    
    printf("Type a character and press Enter: ");
    myChar = getchar();
    
    printf("You entered: ");
    putchar(myChar);
    
    putchar('\n');
    
    return 0;
}
```
## isalpha()
The `isalpha()` function checks whether a character is an alphabetic letter (a-z or A-Z).​ Included in `<ctype.h>`

```c
#include <stdio.h>
#include <ctype.h>

int main() {
    char ch;
    
    printf("Enter a character: ");
    ch = getchar();
    
    if (isalpha(ch)) {
        printf("'%c' is a letter\n", ch);
    } else {
        printf("'%c' is not a letter\n", ch);
    }
    
    return 0;
}
```

| `isupper(c)` | Check if uppercase letter | `isupper('A')` → true |
